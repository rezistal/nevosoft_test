Тестовое задание для игрового программиста С++

Вам необходимо запрограммировать интерактивную визуальную сцену, состоящую
из:
- эффекта салюта - множество отдельных партиклов (хоть точки) движущихся
по различным траекториям;
- нескольких интерактивных UI элементов (кнопок / переключателей и т.п.)
влияющих на характеристики салюта;
- кнопки, вызывающей модальное игровое окно для закрытия сцены с двумя
кнопками: выход, продолжить;
- информационных надписей - fps, количество партиклов, количество
запущенных салютов за сессию.

Требования к салюту:
- должен циклически запускаться на экране с различными вариациями
характеристик (позиции, цвета, траектории);
- салют должен появляться в месте, где кликает пользователь;
- возможность одновременного воспроизведения нескольких эффектов салюта.

Формат готового тестового задания для проверки:
- exe-файл / архив, который запускается на Windows без необходимости
установки доп. ПО и библиотек;
- архив с исходниками / ссылка на репозиторий;
- в выборе среды разработки и подключаемых библиотек мы вас не
ограничиваем, но выполнение тестового задания не должно сводиться только
к банальному вызову готовых решений из сторонних либ: то есть пользоваться
системами частиц из этих библиотек нельзя.

Что будет оцениваться:
- понимание игрового цикла;
- уровень ООП (структура классов для реализации салюта и UI элементов);
- интерактивная составляющая;
- визуальная составляющая.
